
# JABATATA - Sistema de Gestão

Este é o código oficial da Jabatata.

### Como colocar no ar (Vercel):
1. Crie um repositório no GitHub chamado `jabatata`.
2. Suba todos os arquivos desta pasta para lá.
3. No Vercel, clique em "Add New Project".
4. Selecione o repositório `jabatata`.
5. Em "Framework Preset", escolha **Vite**.
6. Clique em **Deploy**.

O link gerado pelo Vercel deve ser cadastrado no Google Cloud Console para o Drive funcionar.
